## Me, Graphically: Movies




A D3.js generated svg which displays all the movies I have watched. Only has viewing functionalities for each movie and sorting by rating, but I am working on sorting by genre. Now that I have a working svg, I don't think that it is the most meaningful visualization. Partly because of how little data I got from Letterboxd with their user data downloads. Nevertheless, I'm still happy that it was a great learning experience with D3.js and svg's. In the future, I might extract and append more data from a movie database to make the visualization more insightful.
